# OS-group-project
README

CS11335 Operating Systems – Course Project
Princess Sumaya University for Technology
King Hussein School of Computing Sciences

Project Title:
Performance Evaluation of CPU Scheduling Algorithms

Implemented Algorithms:
1. First-Come First-Served (FCFS) – Non-Preemptive
2. Shortest-Job-First (SJF) – Non-Preemptive

Programming Language:
Python 3

Project Files:
- fcfs_scheduling      → Implementation of FCFS scheduling algorithm
- sjf_scheduling      → Implementation of SJF scheduling algorithm
- report.pdf   → Final project report

--------------------------------------------------
Program Output
--------------------------------------------------

Each program displays:

- Process execution information
- Arrival time
- Burst time
- Start time
- Finish time
- Turnaround Time (TAT)
- Waiting Time (WT)
- Average Turnaround Time
- Average Waiting Time

--------------------------------------------------
Project Description
--------------------------------------------------

This project evaluates CPU scheduling algorithms used in
Operating Systems. The implemented algorithms are compared
using:

- Average Waiting Time (WT)
- Average Turnaround Time (TAT)

Both algorithms use the same shared process set to ensure
fair comparison of performance.
